How this site is built

This website is a custom-built, server-side rendered application designed to mimic the minimalist philosophy of Bear Blog. It prioritizes speed, zero JavaScript on the client side, and simple HTML.

The Stack

The application is a monolith deployed as a single JAR file.

  • Backend Framework: Spring Boot 4
  • Database: PostgreSQL
  • Template Engine: Thymeleaf (Server-side HTML rendering)
  • Markdown Parser: CommonMark (Atlassian) for converting posts to HTML
  • Security: Spring Security (BCrypt & Form Login)

The Frontend

  • CSS: Water.css (Classless stylesheet)
  • JavaScript: None (0kb bundle size)
  • Fonts: System fonts only (San Francisco, Segoe UI, Roboto)

The source code follows the "Entity-Service-Controller" pattern without DTOs to maintain codebase simplicity.
